# Re-Teck Market Radar — 報表站 / Reports Site

這個 repo **只存放自動產生的市場情報「報表」**（靜態 HTML），給外部瀏覽用。

## 👉 線上瀏覽

**<https://walkeratgh.github.io/reteck-radar-site/>**

- `index.html` — 最新一週報表
- `<日期>.html` — 歷史報表（例如 `2026-06-26.html`）

## 報表內容

一頁式互動報表，右上角可切換 **6 種語言**（EN · 繁中 · 简中 · 日本語 · DE · NL），含 6 個分頁：

| 分頁 | 內容 |
|---|---|
| 市場新聞 | 近一週 ITAD / 資料中心 / 電信產業新聞，依產業×地區篩選 |
| 資料中心策略 | GPU 機隊估計、價值路徑（回收 × 再用 × 二手） |
| 裝置策略 | 機會點、筆電 / 手機市佔 |
| 大宗物料價格 | 金 · 銀 · 銅 · 鋁 · 鎳 · 鋰 · 鈷 · 鉑 · 鈀… 6 個月走勢與分級價 |
| 法規與 ESG | 各地區法規與永續驅動因子 |
| 競爭對手雷達 | 競爭對手動態（威脅 / 機會 / 訊號） |

## 這裡看不到程式

> 本 repo **不含任何原始碼、資料來源或處理邏輯** —— 只有「產生好的成品報表」。
> 報表是由內部工具自動產生並發佈，內容約**每週更新**。

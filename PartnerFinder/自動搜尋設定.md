# 開啟「自動網路搜尋」(Brave Search API)

系統的 **Web Search** 頁面可以自動上網搜尋、把候選公司列出來，每筆按一下就帶入建檔。
要用它，需要一把**免費的 Brave 搜尋金鑰**。整個設定只做一次，約 5 分鐘。

---

## 第 1 步：申請免費金鑰

1. 打開 <https://brave.com/search/api/>，按 **Get started**（會進到 API 管理後台）。
2. 用 Email 註冊 / 登入。
3. 選方案時挑 **Free（免費）** 那一個
   （名稱可能是 “Free” 或 “Data for Search - Free”，免費額度：每月約 2,000 次、每秒 1 次）。
   - 有些情況會要你**綁一張信用卡做驗證**——免費方案**不會扣款**，只是驗證用。
4. 建立訂閱後，到 **API Keys**（或 “Subscriptions”）頁面，**複製那把金鑰**
   （是一長串英數字，例如 `BSAxxxxxxxxxxxxxxxxxxxxxx`）。

## 第 2 步：把金鑰貼進系統

1. 到 `PartnerFinder` 資料夾，用記事本打開 **`appsettings.json`**。
2. 找到這一段：
   ```json
   "Search": {
     "Brave": {
       "ApiKey": ""
     }
   }
   ```
3. 把金鑰貼在**雙引號中間**（不要動到其他符號）：
   ```json
   "Search": {
     "Brave": {
       "ApiKey": "BSAxxxxxxxxxxxxxxxxxxxxxx"
     }
   }
   ```
4. 存檔。

> 🔒 安全提醒：這把金鑰等於你的帳號密碼，**不要**把填好金鑰的 `appsettings.json` 上傳到 GitHub 或傳給別人。

## 第 3 步：重新啟動系統

- 把黑底視窗關掉，重新雙擊 **`run-on-lan.bat`**（Mac 跑 `run-on-lan.sh`）。

## 第 4 步：確認開好了

- 打開 **Settings** 頁面，看「Web Search API connector」那一列，
  應該從灰色 **Not configured** 變成綠色 **Configured**。
- 到 **Web Search** 頁面，輸入例如 `GPU workstation builder Austin Texas`，按 **Search**，
  就會列出網路搜尋結果，每筆右邊有 **+ File as partner** 可以一鍵建檔。

---

## 常見問題

| 狀況 | 解法 |
|---|---|
| Settings 還是顯示 Not configured | 金鑰沒貼對、或沒重新啟動。檢查引號、重開系統 |
| 搜尋出現「Search failed: 401 / 422」 | 金鑰錯誤或未生效；回後台確認金鑰、方案已啟用 |
| 搜尋出現「429」 | 免費額度用完了（每月 2,000 次），下個月重置，或按需求升級方案 |
| 結果的公司名稱怪怪的 | 系統先用網站網址當預設名稱，建檔頁可以自己改成正確公司名 |

---

## 使用流程建議

1. 到 **Keyword Generator** 產生搜尋詞 → 點該詞的 **Auto search**（會直接帶到 Web Search）。
2. 在 **Web Search** 看結果，把像樣的公司按 **+ File as partner**。
3. 建檔頁補上聯絡人、能力勾選 → 存檔，系統自動評分、分級（A/B/C）。

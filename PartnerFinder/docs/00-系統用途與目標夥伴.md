> 這份用最白話說明「**這套系統是拿來做什麼、我們要找什麼樣的公司**」。
> 技術細節請看其他文件；這份給業務／採購／管理層看。

## 一句話

這套系統是一個**夥伴搜尋與評分資料庫**，幫我們有系統地找出並排序
**IT / AI Infrastructure 系統整合商（SI Supplier）**——特別是那些在
**設備租賃**與**服務中小企業（SME）**的日常業務中，**會大量用到 GPU、記憶體（RAM）、
伺服器等硬體**的公司。

英文定位：
> We use this system to find **IT / AI Infrastructure SI Suppliers** who, in their
> routine **equipment leasing** and services to **SMEs**, need **many GPUs, RAM, etc.**

---

## 商業背景：為什麼 Re-Teck 要找這種公司

Re-Teck 的核心是 **ITAD／資料中心資產處理與硬體再利用**（回收、翻新、二手再流通）。
因此我們特別關注「**大量、且持續消耗硬體**」的公司——這種公司對 Re-Teck 有兩種價值：

1. **需求端（買方 / 供貨對象）**：他們在做設備租賃、幫中小企業建置 IT / AI 環境時，
   需要一批又一批的 GPU、RAM、伺服器。Re-Teck 可以**供應翻新／二手硬體**降低他們成本。
2. **合作端（通路 / 夥伴）**：他們是接觸大量中小企業的通路，可成為 Re-Teck 服務與硬體的
   落地夥伴；他們汰換下來的舊設備，也是 Re-Teck 的回收來源。

簡單說：**他們一直在買硬體、也一直在換硬體——正好對接 Re-Teck 的供給與回收兩端。**

## 我們要找的公司長這樣（理想目標輪廓）

**主體：** IT / AI Infrastructure 系統整合商、資料中心服務商、AI 運算建置商。

**加分訊號（越多越好）：**

- 有做 **設備租賃 / 訂閱式硬體（Hardware-as-a-Service）**。
- 主要客戶是 **中小企業（SME）**，會幫客戶「整套建好、交付、維運」。
- 會**建 AI 伺服器 / GPU 工作站**，或部署 **Edge AI / 本地 LLM**。
- 明顯需要 **GPU（NVIDIA / AMD）** 與 **大量記憶體 / 儲存**。
- 具備 **Data Center Smart Hands、IMAC、Break/Fix、機房散熱與電力規劃** 等落地能力。
- 是 **Microsoft / Dell / Cisco / HPE** 的合作夥伴（代表有量、有信任）。

**這些都是系統裡的「能力欄位」與「評分項目」**——勾選越多，分數越高，
系統會自動把它排到 **A 級（優先聯繫）**。

---

## 這套系統怎麼幫我們找到他們

1. **Keyword Generator（關鍵字產生器）**：輸入國家／城市／服務類型，
   自動產生像「GPU workstation provider Austin」「AI server builder Texas」
   「on-prem AI deployment company Dell NVIDIA partner」這種搜尋詞。
2. **Web Search（自動網路搜尋）**：直接把搜尋詞丟上網（Brave API），
   列出候選公司，一鍵建檔。
3. **建檔 + 能力勾選**：把公司的 IT / Data Center / AI 能力逐項記錄
   （含 GPU、RAM/儲存、AI 伺服器、Edge AI、散熱電力等）。
4. **自動評分（100 分）+ 分級（A/B/C）**：系統依能力自動算分，
   A 級優先聯繫、B 級再確認、C 級暫留。
5. **Dashboard / 匯出**：一眼看總覽（各國分布、AI 能力家數、待聯繫等），
   可匯出 CSV / Excel 給團隊跟進。

## 怎麼快速判斷「這家值不值得聯繫」

看系統分數（A/B/C）之外，人工快速檢查這幾點：

| 訊號 | 代表什麼 |
|---|---|
| 網站有寫 **leasing / rental / as-a-service** | 有持續硬體需求 ✅ |
| 主打 **SMB / SME / small business IT** | 客群對，量分散但穩定 ✅ |
| 有 **GPU / AI server / workstation** 字樣 | 需要高階硬體 ✅✅ |
| 是 **Dell / NVIDIA / Microsoft 夥伴** | 有規模、有信任 ✅ |
| 有 **data center / smart hands / IMAC** | 有落地維運能力 ✅ |

## 名詞小抄（給非技術同事）

- **SI（System Integrator，系統整合商）**：幫客戶把伺服器、網路、軟體整套建好的公司。
- **GPU**：繪圖／AI 運算晶片（如 NVIDIA），跑 AI 一定要，很貴、需求大。
- **RAM（記憶體）**：電腦／伺服器的運算記憶體，AI 與虛擬化需要大量。
- **Edge AI**：把 AI 運算放在「現場設備」上跑，而不是雲端。
- **Local LLM / On-Prem AI**：在客戶自己機房裡跑的 AI／大型語言模型。
- **Smart Hands / IMAC / Break-Fix**：機房現場的裝機、搬遷、故障維修等人力服務。
- **Equipment Leasing**：設備租賃——客戶用租的、不買斷，供應商要持續備硬體。
- **SME / SMB**：中小企業。

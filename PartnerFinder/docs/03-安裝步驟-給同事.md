# 03 · 安裝步驟（給同事 · 非技術也能照做）

這份把「從零到能用」的所有步驟集中在一起。跟著做即可。

---

## A. 先決條件（每台電腦做一次）

1. 安裝 **.NET 8 SDK**（免費，微軟官方）：
   <https://dotnet.microsoft.com/download/dotnet/8.0>
   - 下載 **SDK**（不是只有 Runtime），Windows 或 macOS 版，一路下一步裝完。
2. 確認裝好：開「命令提示字元 / 終端機」，輸入 `dotnet --version`，
   看到 `8.x.x` 就成功。

---

## B. 拿到程式碼

**方式 1（最簡單）：** 收到的 ZIP 直接解壓縮，裡面就有 `PartnerFinder` 資料夾。

**方式 2（用 GitHub）：** 到分支頁面按綠色 **Code → Download ZIP**：
```
https://github.com/WalkeratGH/reteck-radar-site/tree/claude/partner-finder-mvp-dotnet-dbe7mj
```

---

## C. 啟動系統

- **Windows：** 進 `PartnerFinder` 資料夾，**雙擊 `run-on-lan.bat`**。
- **macOS：** 終端機執行（第一次先 `chmod +x run-on-lan.sh`）：`./run-on-lan.sh`

看到 `Now listening on: http://0.0.0.0:5080` 代表成功。
**這個黑底視窗要一直開著**，關掉系統就停。

> 手動指令（效果相同）：在 `PartnerFinder` 資料夾開命令列，執行
> `dotnet run --urls "http://0.0.0.0:5080"`

第一次啟動會自動建立資料庫並放入 3 筆美國範例資料。

---

## D. 在「這台電腦」打開

瀏覽器輸入：`http://localhost:5080`
應該就會看到 Dashboard。

---

## E. 在「手機」打開（同一個 Wi-Fi）

1. 查電腦 IP：Windows 開 `ipconfig`，找「**無線區域網路介面卡 Wi-Fi**」（或「乙太網路」）
   底下的 **IPv4 位址**，形如 `192.168.x.x`。
   - ⚠️ 忽略 VMware / VirtualBox / WSL / VPN 那些介面卡的 IP，那些連了沒用。
2. 手機連上**跟電腦同一個 Wi-Fi**（關掉行動數據）。
3. 手機瀏覽器輸入：`http://電腦的IP:5080`（例如 `http://192.168.1.23:5080`）。

**如果手機連不上（連線逾時）：** 幾乎都是 Windows 防火牆擋住。
用「**以系統管理員身分**」開 PowerShell（開始鍵搜尋 powershell → 右鍵 → 以系統管理員身分執行 →
標題列要有「系統管理員」），貼上並 Enter：
```powershell
New-NetFirewallRule -DisplayName "PartnerFinder 5080" -Direction Inbound -Protocol TCP -LocalPort 5080 -Action Allow
```
還不行 → 檢查「手機和電腦同一個 Wi-Fi」、「Wi-Fi 沒有裝置隔離（可改用家用 Wi-Fi 或手機熱點測）」、
「IP 是不是查對那張網卡」。詳見 `手機使用步驟.md`。

---

## F. 開啟「自動網路搜尋」（選用，需免費金鑰）

想讓系統自動上網找公司，需一把免費 **Brave Search API** 金鑰：
1. 到 <https://brave.com/search/api/> 註冊，選 **Free 免費方案**，複製金鑰（像 `BSAxxxx…`）。
2. 用記事本打開 `PartnerFinder/appsettings.json`，把金鑰貼進：
   ```json
   "Search": { "Brave": { "ApiKey": "貼在這裡" } }
   ```
3. 重新啟動系統。到 **Web Search** 頁面就能自動搜尋、一鍵建檔。
   （完整教學：`自動搜尋設定.md`）

> 🔒 **不要**把填了金鑰的 `appsettings.json` 上傳 GitHub 或外傳，金鑰等同密碼。

---

## G. 資料在哪 / 備份 / 重來

- 資料存在 `PartnerFinder/partnerfinder.db` 這個檔（跟著這台電腦，不是雲端）。
- **備份**：複製 `partnerfinder.db` 即可。
- **重來**：關掉系統、刪掉 `partnerfinder.db`，下次啟動會重建 + 放回範例資料。

---

## 常見問題速查

| 狀況 | 解法 |
|---|---|
| 雙擊 bat 視窗閃一下就關 | .NET 8 SDK 沒裝好；或視窗最後有紅字錯誤，截圖找工程師 |
| `localhost:5080` 打不開 | 系統沒跑起來；確認黑底視窗有 `Now listening` |
| 手機連線逾時 | 防火牆（見 E）、不同 Wi-Fi、或 IP 查錯網卡 |
| 自動搜尋 Not configured | 金鑰沒貼對或沒重啟（見 F） |
| 想在外面也能用 | 需雲端部署，找工程師（見 `04-AI-接手開發說明.md`） |

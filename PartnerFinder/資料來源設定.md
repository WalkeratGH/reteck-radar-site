# 資料來源設定(Hunter.io 聯絡人 / SAM.gov 探勘 / 品牌目錄)

> English version: `Data-Sources-Setup.md`

系統除了 Brave 搜尋和 AI 研究,還有三個資料管道。全部**免費起步**。

---

## 1. Hunter.io — 自動補「聯絡人 / 職稱 / Email」

**做什麼:** AI 研究時,若 Email、Contact Person、Contact Title 還是空的,
系統自動用 Hunter.io 查該公司網域的**公開聯絡人**補上(只填空欄)。

**費用:** 免費方案每月約 25 次查詢;付費約 US$34/月起。

**設定(3 分鐘):**
1. 到 <https://hunter.io> 註冊(免費)。
2. 登入後到 **API** 頁面複製 API Key。
3. 貼進 `appsettings.json`:
   ```json
   "Contacts": { "Hunter": { "ApiKey": "你的金鑰" } }
   ```
4. 重啟系統。Settings 頁「Hunter.io contact finder」變成 **Configured**。

之後不用做任何事——按 AI Research 時自動生效。

---

## 2. SAM.gov — 「Discover」探勘頁(美國官方名單)

**做什麼:** 導覽列多了 **Discover** 頁:輸入州別(如 TX)+ 產業代碼(NAICS,
下拉選單已內建 IT 相關代碼),就從**美國聯邦承包商註冊庫**列出公司(名稱、
城市、官網),按 **⚡** 一鍵建檔+AI 研究。

**費用:** 完全免費(美國政府官方 API)。

**設定(5 分鐘):**
1. 到 <https://sam.gov> 註冊免費帳號。
2. 登入 → **Account Details** → 產生 **API Key**。
3. 貼進 `appsettings.json`:
   ```json
   "Discovery": { "SamGov": { "ApiKey": "你的金鑰" } }
   ```
4. 重啟系統。Settings 頁「SAM.gov discovery」變成 **Configured**。

**使用建議:** 州別 + `541512(系統整合商)` 是最對口的組合;
`423430(電腦設備批發)` 常有做租賃的公司。

---

## 3. 品牌官方夥伴目錄(免申請、直接用)

**Keyword Generator** 頁底部新增「Official Partner Directories」卡片,
一鍵打開 Microsoft / NVIDIA / Dell / Cisco / HPE / Supermicro 官方夥伴目錄
與 CRN 五百大名單。

**用法:** 打開目錄 → 篩選 United States + 你要的領域 → 看到合適公司,
複製它的**官網網址** → 貼到系統的 **Web Search** → 按 **⚡ File + AI research**,
系統自動建檔、填欄、評分。

(若某個官方連結改版失效,用旁邊的「search」備援鍵。)

## 金鑰總表(appsettings.json)

| 功能 | 設定位置 | 哪裡申請 | 費用 |
|---|---|---|---|
| 自動網路搜尋 | `Search:Brave:ApiKey` | brave.com/search/api | 每月 $5 抵用(約 1000 次免費) |
| AI 研究/自動填欄 | `Ai:Anthropic:ApiKey` | console.anthropic.com | 每家 ~$0.02-0.1 |
| 聯絡人補齊 | `Contacts:Hunter:ApiKey` | hunter.io | 免費 25 次/月 |
| SAM.gov 探勘 | `Discovery:SamGov:ApiKey` | sam.gov | 免費 |

> 💡 **金鑰建議貼在 `appsettings.Local.json`**(從 `appsettings.Local.example.json`
> 複製一份)而不是 `appsettings.json`。這個檔被 Git 忽略,所以 `git pull` 更新時
> 金鑰不會被覆蓋、也不會上傳。詳見 `用GitHub更新.md` / `Update-via-GitHub.md`。
>
> 🔒 老規矩:任何填了真實金鑰的檔案都不要上傳 GitHub、不要外傳。

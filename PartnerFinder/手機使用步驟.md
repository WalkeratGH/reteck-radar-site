# 在手機上使用 Partner Finder（電腦跑 · 手機同 Wi-Fi 連）

這份是白話步驟。做法是：**電腦把系統跑起來 → 手機連同一個 Wi-Fi → 用手機瀏覽器打開電腦的網址**。
免費、不用任何帳號。缺點是**電腦要開著**、而且**手機要跟電腦在同一個 Wi-Fi**（在外面用不到）。

---

## 第 0 步：先在電腦安裝 .NET 8（只做一次）

- 到 <https://dotnet.microsoft.com/download/dotnet/8.0>
- 下載 **.NET 8 SDK**（Windows 或 macOS 版），一路下一步安裝完成。

## 第 1 步：把程式碼拿到電腦

最簡單：到 GitHub 這個分支頁面，按綠色 **Code → Download ZIP**，解壓縮。
分支網址：
```
https://github.com/WalkeratGH/reteck-radar-site/tree/claude/partner-finder-mvp-dotnet-dbe7mj
```
（會用 git 的話：`git clone` 後 `git checkout claude/partner-finder-mvp-dotnet-dbe7mj`）

程式在 **`PartnerFinder`** 這個資料夾裡。

## 第 2 步：啟動系統

**Windows：** 打開 `PartnerFinder` 資料夾，**雙擊 `run-on-lan.bat`**。
**Mac：** 打開「終端機」，把 `run-on-lan.sh` 拖進去按 Enter（第一次要先 `chmod +x run-on-lan.sh`）。

看到畫面出現 `Now listening on: http://0.0.0.0:5080` 就代表成功了。
**這個視窗要一直開著**，關掉系統就停了。

> 手動指令版（效果一樣）：在 `PartnerFinder` 資料夾開命令列，輸入
> `dotnet run --urls "http://0.0.0.0:5080"`

## 第 3 步：查電腦的 IP 位址

- **Windows：** 開「命令提示字元」輸入 `ipconfig`，找 **IPv4 位址**，長得像 `192.168.x.x`。
- **Mac：** 系統設定 → Wi-Fi → 你的網路「詳細資訊」，看 **IP 位址**（或終端機輸入 `ipconfig getifaddr en0`）。

假設查到是 `192.168.1.23`。

## 第 4 步：手機打開

1. 手機連上**跟電腦同一個 Wi-Fi**。
2. 手機瀏覽器（Safari／Chrome）網址列輸入：
   ```
   http://192.168.1.23:5080
   ```
   （把 IP 換成你第 3 步查到的，後面 `:5080` 要保留）
3. 就會看到 Dashboard，可以正常新增夥伴、產生關鍵字、匯出等。

💡 想像 App 一樣：在手機瀏覽器選「加入主畫面」，桌面就會出現圖示，點一下直接開。

---

## 連不上的話，檢查這幾點

| 狀況 | 解法 |
|---|---|
| 手機顯示連不上 | 確認手機和電腦是**同一個 Wi-Fi**（不是手機 4G/5G） |
| 還是連不上 | 電腦**防火牆**可能擋住。Windows 第一次跑會跳「允許存取」視窗，要按**允許**；或到防火牆放行 5080 埠 |
| 網址打 `https` 打不開 | 要用 **`http://`**，不是 https |
| 忘記 IP | 重跑第 3 步；筆電換 Wi-Fi 後 IP 可能會變 |

---

## 重要：資料存在哪裡？

資料存在**電腦上的一個檔案** `partnerfinder.db`（在 `PartnerFinder` 資料夾裡）。
所以資料是跟著**這台電腦**走的，不是存在手機或雲端。
- 想備份：把 `partnerfinder.db` 複製起來就好。
- 想重新開始：關掉系統、刪掉 `partnerfinder.db`，下次啟動會重建並放回範例資料。

> 如果之後想「在外面也能用、不用一直開電腦」，那就要改成**部署到雲端**——跟我說一聲我再幫你做。
